# AI UI Builder

## 项目简介

本项目是一个基于自然语言驱动的前端系统，通过解析用户输入，将自然语言转换为结构化 UI 配置（JSON），并动态渲染页面组件，实现低代码界面构建能力。

---

## 核心能力

（1）动态组件系统
支持模块的新增、删除与管理，组件基于统一结构进行渲染

（2）配置驱动 UI
页面不依赖静态代码，而是通过 JSON 配置动态生成

（3）拖拽布局
支持模块排序调整，实现灵活的页面布局能力

（4）AI 扩展能力
可接入大模型（如 DeepSeek / GPT），实现自然语言生成 UI 配置

---

## 技术选型

前端框架：React + TypeScript
样式方案：Tailwind CSS
动画支持：Framer Motion
状态管理：React Hooks

---

## 系统架构

自然语言输入 → AI解析 → JSON配置 → 前端渲染 → 页面生成

---

## 如何运行

```bash
npm install
npm run dev
```

---

## 部署方式

（1）将项目上传至 GitHub
（2）导入至 Vercel
（3）构建命令：npm run build
（4）输出目录：dist

---

## 项目亮点

通过“自然语言 → UI配置 → 动态渲染”的方式，实现了 AI 与前端的结合，具备低代码平台的基础能力。
